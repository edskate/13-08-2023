Variáveis são espaços em memória onde podemos alocar algum valor temporariamente.Como O nome diz, O valor de uma variável pode ser alterado.

Após esses pequenos exempos já podemos definir algumas regras pra trabalhar com o javascript com linguagem.

  1. - Sempre que for definir criar uma variável,escreva "Var" antes do nome da mesma.
  2. Isso será bastante útil e seu benificio será mostrado quando falarmos sobre escopo de variável mais á frente;
  3. -Toda linha deve terminar com o sinal de ";"(Ponto e virgula). Assim o interpretador saberá que aquelea linha de código acabou.
  4. - Um código em javascript deve ser colocado entre as TAGS , mas essa não é o foco no momento.
